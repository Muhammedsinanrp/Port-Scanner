# Network Port Scanner

A Python-based multi-threaded network port scanner that detects open ports and running services on a target IP or domain.

## 🚀 Features
- Scan TCP ports from 1 to 1024
- Multi-threaded for faster scanning
- Detects open ports and banner grabbing for services
- Simple CLI interface

## 🛠️ Technologies Used
- Python
- socket library
- threading (via concurrent.futures)
